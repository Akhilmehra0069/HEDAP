Address Management Related Classes

Address Management is made up of several components:

  • Multiple Address objects supported on Accounts
  • Seasonal Address support

The classes involved with Address object support include the following trigger handlers:

  • ADDR_Account_TDTM
  • ADDR_Addresses_TDTM
  • ADDR_Contact_TDTM

The classes involved with Seasonal Address support include the following schedulable class

  • ADDR_Seasonal_SCHED
